Hosting

¿Que es hosting?

Web hosting es un servicio para alojar tus sitios web, para que las personas puedan acceder a él en todo momento a través de diferentes dispositivos siempre que tengan conexión a internet.
Si deseas arrancar un negocio o proyecto, necesitas un web hosting para tu sitio web.

Tipos de Hosting

Hosting Compartido

Dependiendo de los recursos disponibles, permite tener una cantidad variable de dominios y sitios web en una misma máquina.
Sus reducidos costes han hecho posible que muchos profesionales hayan podido llevar su actividad también al mundo en línea.

Hosting Privado

Se le denomina a un server que tiene acceso y recursos privados e independientes, y que se utiliza por un solo cliente, es decir, que no es parte del clásico hosting compartido.
También se caracteriza por tener direcciones IPs dedicadas, es decir que no se comparten con nadie más.

Hosting Dedicado

Es un ordenador comprado o arrendado que se utiliza para prestar servicios dedicados, generalmente relacionados con el alojamiento web y otros servicios en red, en el caso de los servidores dedicados generalmente es un solo cliente el que dispone de todos los recursos de la máquina para los fines que haya contratado el servicio.

Hosting Cloud

Un servidor de nube es un recurso de servidor centralizado y agrupado que se aloja y distribuye a través de una red (generalmente Internet) y al que pueden acceder múltiples usuarios cuando lo necesiten.

Características

1.- Para acceder a la información es necesario contar con conexión a internet.
2.- La información de los sitios web se resguardan en grandes bases que centralizan información de una gran cantidad de dominios.
3.- Los servidores hosting requieren una amplia disponibilidad de espacio y una infraestructura especializada, y son una opción para mantener segura una página y lista para ser ejecutada.
4.- Estos servicios generalmente incluyen la contratación de un dominio web y de una plataforma de correo electrónico.
5.- Al resguardar la información de tu página en estos servidores tienes la certeza de que no la perderás si algo le ocurre a tu equipo.
